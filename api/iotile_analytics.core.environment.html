classmethod SetupScript()[source]¶

Configure iotile_analytics to run in an attended script.

Attended scripts are those that are watched by a user in the command line but not run using jupyter-console. They are run as normal python scripts.

No graphical plotting is allowed since the shell doesn’t allow it and progress bars are displayed in an ascii format.
