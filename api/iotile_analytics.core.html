iotile_analytics.core package¶

High level entry point for iotile_analytics.core package.

Key Functions and Classes¶

AnalysisGroup A top level entry point for selecting and analyzing data.
CloudSession A persistent session to IOTile.cloud.
Environment Configuration class for better interactive computing experience.
ProgressBar A progress bar that automatically chooses the best display method.

Submodules¶

environment Basic configuration for interactive vs script environments.
exceptions Custom exceptions for iotile analytics.
group The AnalysisGroup is the main entry point for analyzing data.
session Simple session management system to save an iotile.cloud token.
stream_series A Pandas DataFrame subclass for dealing with TimeSeries data from IOTile.cloud streams.

Subpackages¶

channels All of the supported ways to connect an AnalysisGroup with data.
interaction Interactivity primitives that work across environments.
utilities A collection of useful numerical utility functions.
