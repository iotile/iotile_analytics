iotile_analytics.core.stream_series module¶

A Pandas DataFrame subclass for dealing with TimeSeries data from IOTile.cloud streams.

Bases: pandas.core.frame.DataFrame

A DataFrame subclass for handle stream timeseries data.

All arguemnts are passed through to the underlying dataframe except for the optional stream and variable arguments that let you set associated metadata and units for this stream.

set_stream(stream)[source]¶

Set the stream metadata for this data series.

This data is what is returned by the iotile.cloud API /api/v1/stream/<slug>/

It contains the stream slug, some metadata and the list of available output units for conversion.

Parameters:stream (dict) – Stream metadata returned from iotile.cloud
