iotile_analytics.interactive package¶

Key Functions and Classes¶

AnalyticsApplication Base class for Bokeh applications built from iotile_analytics objects.
AnalyticsObject Base class for all objects that can be added to an AnalyticsApplication.
BaseViewer A simple interface to plot data.
LogLogViewer A data series viewer with a logarithmic x and y axis.
TimeSelectViewer A graph showing a high level overview of one or more StreamSeries.
show Show an application or analytics object in a notebook.

Submodules¶

app Default implementation for a bokeh application that can show itself in a server or notebook.
viewer_base Base viewer class for interactive plots.
viewer_loglog Helper classes showing interactive timeseries.
viewer_timeselect A widget showing when in time a StreamSeries has data.
