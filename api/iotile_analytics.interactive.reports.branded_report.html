iotile_analytics.interactive.reports.branded_report module¶

A base class for reports that include a nice Arch header and footer.

Summary¶

Classes:

BrandedReport LiveReport base class with fancy header and footer.

Reference¶

class iotile_analytics.interactive.reports.branded_report.BrandedReport(target, source_info, link_url='https://app.iotile.cloud')[source]¶

Bases: iotile_analytics.interactive.reports.report.LiveReport

LiveReport base class with fancy header and footer.

Parameters:
  • target (int) – Whether we are targetting a hosted or unhosted environment so that we generate the appropriate content.
  • source_info (dict) – A dictionary of source information obtained from an analysis group that allows us to setup the correct header and footer information.
  • link_url (str) – An optional direct link to the object that we are showing a report about in app.iotile.cloud. If not passed it defaults to the generic start page of app.iotile.cloud.
