iotile_analytics.interactive.reports.handlers package¶

Helper classes to manage saving report output.

The classes in this package are used by analytics-host to properly save the files produced by an AnalysisTemplate either locally, as part of a zip file or by pushing them directly to a remote cloud server.

Key Functions and Classes¶

FileHandler Base class for all FileHandlers supported by analytics-host.
LocalDiskHandler Base class for all FileHandlers supported by analytics-host.
StandardOutHandler A simple FileHandler that just writes all files to stdout.
StreamingWebPushHandler Asynchronous handler that pushes each file to the cloud as generated.
WebPushHandler Push a report from local disk to the cloud.
ZipHandler A file handler that zips up all of the files.

Submodules¶

base_handler Base class for all file handlers supported by analytics-host.
file_handler Basic file handler that just saves files to disk.
stdout_handler A simple handler that just writes its contents to stdout.
streaming_web_push_handler Report File Handler that streams each file to the cloud as generated.
web_push_handler A basic web push handler that pushes a report to the cloud after it has been written locally.
zip_handler A file handler that zips up all of the files.
