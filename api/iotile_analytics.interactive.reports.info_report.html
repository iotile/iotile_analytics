iotile_analytics.interactive.reports.info_report module¶

Basic LiveReport that just prints information about an AnalysisGroup as a txt file.

class iotile_analytics.interactive.reports.info_report.SourceInfoReport(group, streams=False)[source]¶

Bases: iotile_analytics.interactive.reports.analysis_template.AnalysisTemplate

A basic AnalysisTemplate that just prints source information.

If you pass the argument streams as True, the report will also include a list of all data streams in the AnalysisGroup. This AnalysisTemplate can be directly viewed on stdout without needing to be saved to a file but can also optionally generate a text file (.txt) with the metadata that it prints.

Parameters:
  • group (AnalysisGroup) – The group that we wish to analyze.
  • streams (bool) – Include stream summary information as well in the report. This defaults to False if not passed.
run(output_path)[source]¶

Render this report to output_path.

If this report is a standalone html file, the output path will have .html appended to it and be a single file.

If this report is not standalone, the output will be folder that is created at output_path.

If bundle is True and the report is not standalone, it will be zipped into a file at output_path.zip. Any html or directory that was created as an intermediary before zipping will be deleted before this function returns.

Parameters:output_path (str) – the path to the folder that we wish to create.
Returns:
The path to the actual file or directory created. This
may differ from the file you pass in output_path by an extension or the addition of a subdirectory.
Return type:str
