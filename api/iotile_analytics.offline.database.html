Reference¶

class iotile_analytics.offline.database.OfflineDatabase(path=None)[source]¶

Bases: object

An offline database.

If you pass an existing file, this database will open that file. If you pass a non-existing file, this database will create a new database. If you pass nothing, a new in-memory database will be created that will not be backed by file storage.

Parameters:path (str) – The path to the database file that we want to create or open. If None if passed (the default), a new, in-memory database is created that will be lost when the program is exited.
